LUCIBOX

Lucibox est un projet reliant la culture Maker D.I.Y avec le monde des machines musicales
Il permet au plus grand nombre, de construire soi-même des machines audio ( sampler, looper, pédale d'effet), à partir des cartes Arduino et Raspberry Pi.

Retrouvez le projet sur le site UN DES SENS
Retrouvez le projet sur la page GITHUB

MA PART DANS CE PROJET

tout ( ou presque )

  • Création du logiciel basé sur Pure data
  • Création du modèle électronique
  • Portage du code sur plateforme Raspberry pi
  • Benchmark des carte sons utilisables
  • Optimisation audio ( linux temps réel, calculs de latences
  • Documentation sur github
  • Tutoriel vidéo youtube
  • Réalisation d'ateliers ( collège, lycée, université, médiathèque )
  • Atelier Event (maker fair, festival yeah, open bidouille camp)
